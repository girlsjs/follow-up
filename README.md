# Girls.js Follow-Up Repository

## Installation

```bash
git clone git@github.com:girlsjs/follow-up.git
cd follow-up
yarn install
yarn gitbook-setup
```

## Run
```bash
yarn start
```

## Build (for deploy)
```bash
yarn build
```
